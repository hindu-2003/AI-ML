### What is function?

If a group of statements is repeatedly required then it is not recommended to write these statements everytime seperately.
We have to define these statements as a single unit and we can call that unit any number of times based on our requirement without rewriting.

This unit is nothing but function.

The main advantage of functions is code Reusability.

A function is a block of code which only runs when it is called.

You can pass data, known as parameters, into a function.

A function can return data as a result.

**Note:** In other languages functions are known as methods,procedures,subroutines etc

## Recursive Functions

**A Function that calls itself is known as Recursive Function.**
if we want to 
factorial(n)
factorial(n)=n*(n-1)*(n-2)*(n-3)*...1---->normal

factorial(n)=n*factorial(n-1)----->Recursive

**1.We can reduce length of the code and improve readability**

**2.We can solve very complex problems also very easily.**

```
def factorial(n):
    if n==0:
        result = 1
    else:
        result= n*factorial(n-1)
    return result
print(factorial(5))

120

###  Example usage: Generate a power bill for 200 kWh of usage
generate_power_bill(200)

```
def calculate_usage_charge(usage):
    # Assume a rate of $0.10 per kilowatt-hour
    rate_per_unit = 3.50
    return usage * rate_per_unit

def calculate_tax(subtotal, tax_rate):
    return subtotal * tax_rate

def generate_power_bill(usage):
    # Constants
    tax_rate = 00.9

    # Calculate usage charge
    usage_charge = calculate_usage_charge(usage)

    # Calculate subtotal before tax
    subtotal = usage_charge

    # Calculate tax
    tax = calculate_tax(subtotal, tax_rate)

    # Calculate total bill
    total_bill = subtotal + tax

    # Print the bill details
    print("Power Bill Statement")
    print("--------------------")
    print(f"Usage Charge: ${usage_charge:.2f}")
    print(f"Tax: ${tax:.2f}")
    print("--------------------")
    print(f"Total Bill: ${total_bill:.2f}")

output:

tasks = []

def add_task(task):
    tasks.append({"task": task, "completed": False})
    print("Task added successfully!")

def view_tasks():
    print("\n===== Task List =====")
    for index, task_info in enumerate(tasks, 1):
        status = "Completed" if task_info["completed"] else "Not Completed"
        print(f"{index}. {task_info['task']} - {status}")
    print("=====================")

def mark_completed(index):
    if 1 <= index <= len(tasks):
        tasks[index - 1]["completed"] = True
        print("Task marked as completed!")
    else:
        print("Invalid task index!")

def delete_task(index):
    if 1 <= index <= len(tasks):
        deleted_task = tasks.pop(index - 1)
        print(f"Task '{deleted_task['task']}' deleted successfully!")
    else:
        print("Invalid task index!")

def main():
    while True:
        print("\n===== Task Manager =====")

            view_tasks()

        elif choice == 4:
